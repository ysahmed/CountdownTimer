About
---
Just a simple countdown timer app. __\[WORK IN PROGRESS\]__

This project is built by following MVVM architecture.  
The purpose of this project experiment with the following Android and Jetpack components:

- Service
- AlarmManager
- BroadcastReceiver
- Local notification
- Theme
- Color
- Navigation
- Room database
- Livedata
